## Simon Says Memory Game

An interactive **Simon Says memory game** built using **HTML5, CSS3, and JavaScript**.  
The game generates a sequence of colors that the player must remember and repeat correctly. With each level, the sequence becomes longer, making the game more challenging.

This project demonstrates core front-end development concepts such as **JavaScript logic building, DOM manipulation, event handling, and dynamic user interaction**, while providing a fun and engaging gameplay experience.

### Features
- 🎮 Interactive color sequence gameplay
- 📈 Increasing difficulty with each level
- 🧠 Improves memory and concentration
- ⚡ Dynamic DOM updates using JavaScript
- 🎨 Responsive and clean UI design

### Tech Stack
- **HTML5** – Structure of the game
- **CSS3** – Styling and animations
- **JavaScript** – Game logic and interactions

### How to Play
1. Press any key to start the game.
2. Watch the sequence of colors displayed.
3. Repeat the same sequence by clicking the buttons.
4. The sequence grows longer each level.
5. If you click the wrong color, the game ends.

### Learning Outcomes
- JavaScript event handling
- DOM manipulation
- Game state management
- Logic building
